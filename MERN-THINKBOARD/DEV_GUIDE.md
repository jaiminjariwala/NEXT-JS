# Development Guide - Quick Reference

## Commands Reference

| Command | Purpose | When to Use |
|---------|---------|-------------|
| `npm run dev:backend` | Start backend dev server | Development |
| `npm run dev:frontend` | Start frontend dev server | Development |
| `npm run build` | Build for production | Before deploying |
| `npm run start:prod` | Test production build | Before deploying |
| `npm run start` | Start backend (no NODE_ENV) | Not recommended |
